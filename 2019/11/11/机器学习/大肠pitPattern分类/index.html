大肠pit pattern的分类

Posted on 2019-11-11

“pit”是什么？

pit在医学辞典中是“表面的小窝”的意思。在大肠肿瘤的“pit”是大肠粘膜陷凹的开口部分，它会随着周围粘膜的形态的变化为产生各种各样的形态学的变化，应用这些形态来进行诊断就是所谓的“pit pattern”诊断，首先我们来看“pit pattern”的分类。

pit pattern 的分类？

  • I型（圆形）
  • II型（星芒型）
  • III型
    • $III_s$型（小类型圆形）
    • $III_L$型（大类型圆形，管状型）
  • IV型（脑回状）
  • V型
    • $V_I$型（轻度不规整，高度不规整，pit的荒废破坏）
    • $V_N$型（无构造，~DR显著）

普通内镜观察下的pit pattern

  • I型

    呈现正常的类圆形的pit pattern，不规则的腺管和小型的$III_L$很难区分

  • II型

    呈现星芒状的交大的pit pattern，只用普通内镜难以和$III_L$型区分开

  • $III_L$型

    比正常的pit大的管状型的pit组合而成，喷洒靛蓝胭脂图像，结晶紫染色图像也可以观察到，但是不使用放大内镜难以和$V_I$区分

  • $III_s$

    小型管状或类圆形的pit pattern，比正常腺管开口小的pit，很难观察，经常被认为是无构造的pit（$V_N$型）。因此放大倍数低的时候很难判定

  • IV型

    沟纹型，树枝型，脑回状的pit pattern像，这些用普通的内镜都是可以观察的，但是是否伴有$V_I$型 和$V_N$型，还需要更详细的观察

  • $V_I$型

    大小不等，左右排列非对称的pit像

  • $V_N$型

    病变表层可以看到间质反应

总结

  • V型的判断，用普通的内镜不能很好的判断，必须要用放大内镜观察。
  • 在鉴别$V_I$型和$III_L$型以及最重要的$V_N$型时，用普通内镜观察比较困难
  • 用普通内镜可以判断I型、$III_L$、IV，但是V型，特别是$V_I$型和$V_N$型精确度较高的诊断，对于肿瘤的良恶性判断都是必不可少的
